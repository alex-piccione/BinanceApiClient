# Alex75.BinanceApiClient

Simple interface to facilitate the use of Binance API.  


## XRP

Binance does not offer the pair _XRP/USD_ nor _XRP/EUR_.  
XRP can only be exchanged using:  
- XRP/BTC
- XRP/ETH
- TRX/XRP
- ZXC/XRP
